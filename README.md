# A92_c51_public项目说明
## 简介
本仓库是作者第的第一个8051单片机项目，用于学习8051单片机和小尺寸显示屏。接线方法请见具体版本文件夹中的README文档。
程序运行平台为STC89C516RD+或STC11F60XE，请使用STC-ISP工具进行烧录。
>目前仓库中仅含有1.3.1_public版本的HEX文件，作者有空时可能会上传其他版本和源码。
